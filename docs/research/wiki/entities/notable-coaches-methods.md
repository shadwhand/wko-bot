# Notable Coaches & Methodologies

Key coaching figures in power-based cycling training, their philosophies, and how they compare.

Evidence levels: **[R]** = Research-backed, **[E]** = Experience-based, **[O]** = Opinion.

---

## 1. Dr. Andrew Coggan -- The Scientific Foundation

### Background

- **Internationally recognized exercise physiologist** with numerous scientific publications
- Former national-caliber masters cyclist and time trial record holder
- USA Cycling Sport Science Award (2006); US Olympic Committee Doc Councilman Award Finalist
- Created the foundational metrics that the entire power-training ecosystem is built on

### Key Contributions

| Contribution | Impact |
|-------------|--------|
| **Normalized Power (NP)** | Accounts for variability in ride power; physiologically weighted average |
| **Training Stress Score (TSS)** | Quantifies training load per session relative to FTP |
| **Intensity Factor (IF)** | Ratio of NP to FTP; describes how hard a ride was |
| **Performance Management Chart (PMC)** | CTL/ATL/TSB framework for tracking fitness/fatigue/form |
| **Classic 7-Zone Power Training Levels** | The 2001 anchor-point system defining training zones relative to FTP |
| **Power Profiling** | Categorizing athletes by peak power at key durations (5s, 1min, 5min, FTP) |
| **Quadrant Analysis** | Plots force vs velocity to characterize pedaling demands |
| **WKO4/5 Power-Duration Model** | Mathematical model of the entire power-duration relationship |
| **iLevels** | Individualized training levels derived from the PD model |
| **Auto-Phenotyping** | Automatic athlete phenotype classification from PD curve shape |

### Philosophy

- **Data-first, science-grounded** -- everything built on validated mathematical models [R]
- **Anchor-point approach** -- FTP as the baseline, with zones defined relative to it [R]
- **Acknowledged limitations of his own system** -- classic zones work well below threshold but break down above it due to individual variability [R]
- **Cost-benefit evolution** -- added iLevels when the PD model made individualization worth the complexity [R]

### Limitations & Criticisms

- The classic 7-zone system uses percentages above FTP that assume similar physiology across athletes -- a match sprinter and a time trialist can differ by 4x in short-duration relative power [R]
- TSS/CTL framework is widely misinterpreted as a direct "fitness" measure (EC explicitly objects to CTL=fitness) [E]
- PMC does not account for training specificity, rest quality, or life stress [E]

## 2. Tim Cusick -- Individualized Training with WKO

### Background

- WKO4/5 Product Manager and key architect of the platform's coaching workflows
- TrainingPeaks coaching educator; co-authored iLevels article with Coggan
- Coaches athletes at all levels using WKO's individualized analytics

### Key Contributions

| Contribution | Impact |
|-------------|--------|
| **iLevels coaching workflow** | Practical application of Coggan's iLevels for daily coaching |
| **Optimized Intervals in WKO** | WKO's interval prescription based on individual PD curve |
| **Strengths & Limiters analysis** | Systematic identification using PD curve profile charts |
| **Power Duration Metrics History** | Tracking micro-changes in fitness through PD curve evolution |
| **Time-intensity specificity** | Individualizing not just power but also duration targets per level |

### Philosophy

- **"The principle of individuality dictates that decisions concerning training should be made with each individual athlete in mind"** [E]
- **Moves beyond percentage-based zones** -- classic systems initialize training but fall short in individualization, particularly above threshold [E]
- **Dose-response precision** -- training should be prescribed and tracked as specific doses targeting specific responses [E]
- **Micro-change detection** -- PD model updates with each file upload, capturing fitness changes that formal testing would miss [E]
- **Extensive vs intensive intervals** -- distinguishes between building endurance at a level (longer @ lower end) vs building power at a level (shorter @ higher end) [E]

### Practical Workflow (from WKO4 Training Plan Construction)

1. Identify goal race demands using past race files (FRC match burn rate, duration, phenotype requirements)
2. Assess athlete strengths/limiters using PD curve profile and W/kg comparison charts
3. Match strengths/limiters to event demands
4. Use historical PMC data to determine optimal CTL and TSB targets
5. Establish CTL ramp rate parameters from prior season analysis
6. Prescribe intervals using WKO optimized intervals for each training phase
7. Track results via PD curve changes, not isolated formal tests

## 3. Kolie Moore -- Empirical Cycling (Data-First Coaching)

### Background

- Host of the Empirical Cycling podcast (TMT, Perspectives, Watts Doc series)
- Coaches a wide range of athletes from amateur to elite
- Known for rigorous evidence standards and willingness to challenge conventional wisdom
- The primary source for the EC master reference document underpinning this platform

### Key Contributions

| Contribution | Impact |
|-------------|--------|
| **FTP = MLSS framework** | Corrected widespread "1-hour power" misconception; FTP is 30-70 min depending on athlete |
| **Durability framework** | Formalized kJ/kg methodology for measuring fatigue resistance |
| **Fueling revolution** | Advocated 60-90g/hr carbs when 25g/hr was standard |
| **TTE as key metric** | Time to exhaustion at FTP as progression target |
| **FTP training decision tree** | Structured approach: season point, training age, opportunity cost |
| **Anti-panic protocol** | 8+ weeks = add volume first; <4 weeks = pivot goals |
| **Strength as a skill** | Neural maintenance with heavy singles, minimal volume |
| **Rest-week philosophy** | Athlete requests rest = ALWAYS grant; motivation as physiological signal |

### Philosophy

- **Evidence hierarchy: research > coached experience > opinion** -- explicitly labels all claims with [R]/[E]/[O] tags [E]
- **Individual variation dwarfs protocol effects** -- -20% to +30% individual variation vs ~1-2% distribution effects [R]
- **"No single training intensity distribution is statistically superior"** -- the IPD meta-analysis showed no meaningful difference (Persp-38) [R]
- **"Zone 2 has nothing special about it"** -- total volume matters more than zone specificity (TMT-69) [E]
- **Progressive overload before protocol change** -- exhaust simple duration progression before switching (TMT-44, 45) [E]
- **Goodhart's Law awareness** -- warns against over-indexing on any single metric [E]
- **Auto-regulation over rigid compliance** -- 100% plan compliance is a warning sign (TMT-52) [E]
- **"Written in sand"** -- training plans should be flexible, not dogmatic (TMT-73) [E]

### Key Contrarian Positions

| Conventional Claim | Moore's Position | Evidence |
|-------------------|-----------------|----------|
| Zone 2 is special | Total volume > zone specificity | TMT-69 [E] |
| More training always better for time-crunched | May need rest, not more intervals | TMT-72 [E] |
| CTL = fitness | Specifically rejected | TMT-68 [E] |
| Train low improves performance | Molecular dead end | WD-54, 60 [R] |
| Durability is the missing piece | Need sufficient absolute power first | WD-60 [E] |
| 30/15s are best VO2max training | No VO2max superiority; gains likely W' | WD-55 [R] |

## 4. Joe Friel -- Classical Periodization

### Background

- Author of "The Cyclist's Training Bible" (first published 1996, now in 5th edition)
- Co-founder of TrainingPeaks (Peaksware)
- One of the earliest advocates of structured, periodized training for endurance athletes
- Coached numerous elite and age-group athletes over 30+ years

### Key Contributions

| Contribution | Impact |
|-------------|--------|
| **Periodization framework** | Formalized base/build/peak/race/transition periods for cycling |
| **Annual Training Plan (ATP)** | Structured season planning tool (now integrated in TrainingPeaks) |
| **Training Bible methodology** | Made periodized training accessible to self-coached athletes |
| **Transition period philosophy** | Advocated structured off-season rest and reflection |
| **Heart rate zone training** | Early structured approach before power meters were widespread |

### Philosophy

- **Classical linear periodization** -- base -> build -> peak -> race -> transition [E]
- **Systematic approach** -- structured mesocycles, microcycles, and macrocycles [E]
- **Self-coached athlete empowerment** -- made coaching concepts accessible via books and TrainingPeaks [E]
- **Data tracking as habit** -- advocated consistent workout logging long before it was common [E]
- **Recovery as a training phase** -- transition periods are not wasted time but essential adaptation [E]

### Comparison to Modern Approaches

| Aspect | Friel (Classical) | Moore (EC) |
|--------|-------------------|------------|
| Periodization model | Linear base-build-peak-race | Flexible blocks; "written in sand" |
| Testing | Regular formal tests | Rolling PD curves; formal tests every 3-4 months max |
| Zone system | HR-based initially, then power zones | iLevels or simplified 3-zone + power |
| Recovery philosophy | Scheduled recovery weeks | Auto-regulated; athlete requests always honored |
| Intensity distribution | Moderate polarization | "No distribution is statistically superior" |
| Self-coaching | Encouraged via books/ATP | Encouraged via data literacy |

## 5. Hunter Allen -- Power Meter Popularizer

### Background

- Co-author with Coggan of "Training and Racing with a Power Meter" (the definitive book, 3 editions)
- Peaks Coaching Group founder
- USA Cycling certified Level 1 coach; former professional cyclist
- Played a major role in making power-based training accessible to the mass market

### Key Contributions

| Contribution | Impact |
|-------------|--------|
| **Training and Racing with a Power Meter** | The foundational book that brought Coggan's science to practitioners |
| **Power profiling methodology** | Practical application of 5s/1min/5min/20min peak power categories |
| **Sweet spot training popularization** | Advocated 88-94% FTP as time-efficient training zone |
| **WKO development** | Contributed to WKO software evolution from early versions through WKO4 |
| **Coaching education** | Trained hundreds of coaches through Peaks Coaching Group |

### Philosophy

- **Practical application of science** -- translates Coggan's research into actionable coaching [E]
- **Power meter as the essential training tool** -- advocacy that drove mass adoption [E]
- **Sweet spot as a cornerstone** -- time-efficient training for age-group athletes [E]
- **Systematic testing and profiling** -- 5s/1min/5min/FTP testing protocol for strengths/limiters [E]

### EC Perspective on Sweet Spot

Moore acknowledges sweet spot as a useful tool but notes it becomes time-inefficient as fitness improves -- well-trained athletes need 2+ hour sessions for equivalent stimulus (TMT-44) [E]. Sweet spot "is a tool, not a philosophy" [E].

## 6. Comparative Matrix

### Methodology Comparison

| Dimension | Coggan | Cusick | Moore (EC) | Friel | Allen |
|-----------|--------|--------|------------|-------|-------|
| Primary medium | Research papers, WKO | WKO workflows | Podcast, coaching | Books, TrainingPeaks | Book, coaching |
| Zone system | Classic 7-zone | iLevels (9 zones) | Flexible; 3-zone + power | Classic power/HR zones | Classic 7-zone |
| Individualization | PD model | iLevels + PD model | Individual variation emphasis | Zone-based | Power profiling |
| Periodization | Not prescriptive | Block-based via WKO | Flexible blocks, anti-rigid | Linear classical | Linear with flexibility |
| Key metric | NP, TSS, IF | PD curve shape changes | FTP, TTE, durability (kJ/kg) | CTL (PMC) | Peak power profiles |
| Evidence standard | Peer-reviewed + applied | Applied + data-driven | Explicitly tagged [R]/[E]/[O] | Coaching experience | Applied science |
| Sweet spot view | Valid zone | Trainable via iLevels | Tool, not philosophy | Useful intensity | Cornerstone approach |
| Recovery approach | Not primary focus | Data-driven monitoring | Auto-regulation, always grant rest | Scheduled | Structured |

### When to Use Which Approach

| Situation | Best Fit | Rationale |
|-----------|----------|-----------|
| New to power training | Allen/Coggan | Foundational concepts, accessible via book |
| Self-coached with data | Friel/TrainingPeaks | Structured season planning, ATP tool |
| Plateau after 2+ years | Moore/EC | Challenge assumptions, auto-regulate, address durability |
| Individualized coaching | Cusick/WKO5 | iLevels, optimized intervals, micro-tracking |
| Research questions | Coggan | Primary science source |
| Ultra-endurance focus | Moore/EC | Durability framework, fueling, TTE emphasis |

## 8. Platform Integration Hints

### How the Platform Synthesizes These Approaches

The analytics platform draws primarily from:

1. **Coggan's mathematical foundations** -- PD model, NP, TSS, IF calculations form the data layer
2. **Cusick's individualization** -- iLevels and optimized intervals for prescription
3. **Moore's evidence framework** -- [R]/[E]/[O] tags, durability methodology, contrarian checks
4. **Friel's periodization structure** -- season planning and ATP concepts
5. **Allen's practical accessibility** -- power profiling and sweet spot as entry points

### Conflict Resolution Protocol

When coaches disagree (common above threshold):
- **Tag the evidence level** of each position
- **Present both perspectives** with sources
- **Let the athlete's data decide** -- individual PD curve and response to training trump theoretical debates
- **Default to the more conservative position** when evidence is ambiguous
