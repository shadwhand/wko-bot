# Q4 (Augmented): Design a 3-week VO2max block for me. I have 10 hours/week available.

Here is a 3-week VO2max block using a 2:1 load/recovery structure -- two hard weeks followed by a mandatory rest week, per EC TMT-55 guidelines for VO2max blocks [E].

## Weekly Template (Weeks 1-2)

| Day | Session | Prescription | Duration |
|-----|---------|-------------|----------|
| Mon | Rest | Off the bike | 0h |
| Tue | VO2max intervals | See progression below | 1.5h total |
| Wed | Endurance + sprints | IF < 0.65; 4x10s sprints in first 20 min | 1.5h |
| Thu | Threshold maintenance | 2x15 min @ 95-100% FTP, 5 min rest | 1.5h total |
| Fri | Endurance | IF 0.50-0.65, RPE 2-3/10 | 1.5h |
| Sat | VO2max intervals | See progression below | 1.5h total |
| Sun | Long endurance | IF 0.55-0.65 | 2.5h |

Total: ~10 hours. Two VO2max sessions per week (the research-supported maximum; Seiler 2010 [R]), one threshold maintenance session (TMT-60 [E]), and endurance fill.

## VO2max Interval Progression

Use classic long intervals -- the best starting point for an ultra-endurance athlete with a big volume base (AMA [E], protocol selection guidance from vo2max-training wiki).

| Week | Tuesday | Saturday |
|------|---------|----------|
| 1 | 4x3 min @ 110% FTP / 3 min easy | 4x3 min @ 110% FTP / 3 min easy |
| 2 | 5x3 min @ 110% FTP / 3 min easy | 4x4 min @ 108% FTP / 3 min easy |
| 3 | REST WEEK | REST WEEK |

Target RPE: 9-9.5/10 on the final rep (TMT-49 [E]). Use high cadence, 100-120 RPM, to reduce muscular fatigue (WD-55 [R+E]). Total time at intensity per session stays around 12-20 min (WD-55 [E]).

## Week 3: Mandatory Rest

- Volume: 40-60% of normal (~4-5 hours), all rides IF < 0.50 (TMT-55 [E])
- Days 1-2: very easy spin or off
- Days 3-5: short endurance rides, 60-90 min
- Day 6-7: gating test -- sprint power + 5 min at FTP; if both feel good, resume loading (TMT-55 [E])

## Key Execution Rules

1. **Fuel the work.** No caloric restriction during VO2max blocks -- high-intensity work requires full glycogen stores (WD-54, Persp-41 [R]).
2. **36-48 hours between VO2max sessions** (Santos, TP [R]) -- the Tue/Sat spacing provides this.
3. **Progressive overload before protocol change** -- add one rep or extend interval duration before switching to 30/15s or micro-bursts (TMT-52 [E]).
4. **Monitor warning signs:** HRV depression >7 days, persistent low motivation, failed session + failed retest 2-3 days later = immediate rest (TMT-55 [E]).
5. **Retest after the block:** check 5-min peak power as your VO2max proxy. A >5% gain signals adaptation (Simone, TP [E]).
